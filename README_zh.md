## 背景
  为什么开发mits？原因是我每天需要喝大量的水，但是我经常会忘记喝水这件事。所以我尝试寻找一些工具来提示我喝水。但是很遗憾，我找了很久也没有找到合适的。因为，我决定开发mits。
## 使用方式
  首先，你需要下载你需要的发布版本，然后解压它；然后，将解压后的文件添加到Chrome插件即可。
  
  ***具体步骤***

  1. 打开浏览器，在地址栏输入`chrome://extensions`导航到浏览器扩展页面
  2. 打开开发者模式
  3. 点击`加载已解压的扩展程序`将文件夹导入

  最后，你就可以使用mits了
